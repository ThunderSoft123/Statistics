1 boot.img烧写
		sudo adb reboot bootloader
		sudo fastboot devices
		sudo fastboot flash boot boot.img
2 camera.default.so烧写
		删除 camera.msm8996.so 在目录system/lib/hw下
		然后在将camera.defalut.so push system/lib/hw
3 mediaserver 烧写
		删除 system/bin目录下的 mediaserver
		然后在push 本目录下的 mediaserver
		adb push mediaserver system/bin
4 adb reboot

5 Camera2.apk 的烧写
		adb install -r Camera2.apk
		adb shell am start -n com.android.camera2/com.android.camera.CameraActivity
